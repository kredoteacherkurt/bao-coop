System Operation Guide

Simulation Workflow

  1. Register: Go to signup.html and create a test user.
  2. Login: Access the Member Portal via login.html.
  3. Transact: As a member, apply for a "Loan" or "Savings" product in the dashboard.
  4. Approve: Return to this Admin Dashboard. The request will appear in the Transaction Requests table.
  5. Result: Click Approve (Green Check). Refresh the Member Portal to see the updated balance instantly.

Technical Notes

  • Data Storage: All data is saved in your browser's LocalStorage. It acts like a database but lives in your specific browser.
  • Video Management: To change the homepage video, place your MP4 file in the folder: assets/mp4/. Then, update the filename in the Editor above.
  • Resetting: Use the "Wipe History" or "Delete All Users" buttons to clear the simulation and start fresh.
